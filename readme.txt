I use vs code as an editor so here is an instruction about how to start a live server using vs code and correctly load this lab

 Using Live Server on Visual Studio Code
The live server extension launches a local development server with a live reload feature for static & dynamic pages. 
Step 1: Click on the extensions option on the left side in Visual Studio Code.
Step 2: Search Live Server in the search box and install it.
Step 3: Click on Go live present at the bottom.
Step 4: The default browser will get opened automatically once you click on the go live.

in the file explorer(use "ctrl+b" to open file explorer), right-click the RenderProject.html file,
then select Open with live server
then this lab will be open in your default web browser;

R - roll clockwise

u - move light up   
j - move light down     
i - move light left  
k - move light right
o - move light forward 
l - move light back

s or S - move robot back

d or D - move robot right

t - show texture
